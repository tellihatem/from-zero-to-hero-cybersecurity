# From Zero to Hero — Cybersecurity & Hacking Roadmap

This repository documents my **personal journey from beginner to master in cybersecurity and ethical hacking**.  
It contains everything that helped me learn, practice, and improve my skills over time.

## What’s Inside

- **12-Month Cybersecurity Roadmap:** A structured path from Linux fundamentals to advanced Red Team, Blue Team, and Blockchain security.
- **Lab Writeups:** Step-by-step reports of machines I have hacked, including methodologies, screenshots, and lessons learned.
- **Scripts & Tools:** Python, Bash, and PowerShell scripts I created to automate tasks, enumeration, and exploitation.
- **Templates:** LaTeX templates for Red Team reports, Blue Team incident reports, and general lab writeups.
- **Resources & References:** Cheatsheets, diagrams, links, and materials that supported my learning journey.

## Purpose

The goal of this repository is to:

1. Track and document my personal **learning progress**.
2. Showcase **hands-on skills** in cybersecurity and ethical hacking.
3. Provide a **reference point** for others learning cybersecurity.
4. Keep all my **labs, scripts, and resources organized** in one place.
